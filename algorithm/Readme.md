
# 数据结构与算法

这是 CS 的一个核心课程，非常重要。

## 如何能学好

知乎上关于[如何学好数据结构？](https://www.zhihu.com/question/21318658)有非常多的回答，都很实用。总结一下就是：

1. 学习要有层次性，要多学几遍，每一遍的目标是不一样的
2. 学习要成体系，把零碎的知识点整合成一个稳固的知识体系
3. 学习理论，达到深度的理解，结合实际题目或者应用场景多练习
4. 需要刷题，要刷好几遍，一遍是不行的
5. 每天都要练习，日积月累，坚持下去
6. 动手写代码，动手写代码，动手写代码
