# 数据结构 - 堆

### 堆

什么是堆？

1. 堆是一个完全二叉树
2. 堆中的每个节点的值都必须大于等于（或者小于等于）其子树节点的值

堆中的每个节点都大于等于其子树节点的值，这种堆叫大顶堆；堆中的每个节点都小于等于其子树节点的值，这种堆叫小顶堆
