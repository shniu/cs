# 查找

### 二分查找

* 最常规的 二分查找 算法实现 （两种实现方式：while loop 和 递归\)
* 找不包含重复元素的有序数组旋转后的最小值
* 找包含重复元素的有序数组旋转后的最小值
* 有序数组中查找第一个等于给定值的位置（有序数组可能有重复元素）
* 有序数组中查找最后一个等于给定值的位置（有序数组可能有重复元素）
* 有序数组中查找第一个大于等于给定值的位置（有序数组可能有重复元素）
* 有序数组中查找最后一个小于等于给定值的位置（有序数组可能有重复元素）
